# 8.5 Validator Node Reward

### To reward those machines that provide the computing power, we set incentive rules

| Entries          | Point |
| ---------------- | ----- |
| 1 day validation | 5     |
| 2 days           | 10    |
| 1 week           | 15    |
| 1 month          | 20    |

* 5\~10 points  5000 ETX
* 15\~20 points 10000 ETX
* 20\~30 15000 ETX
* 30\~40 20000 ETX
* 40+ points 30000 ETX

### Note:95% SLA
